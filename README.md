## express 项目架构

|目录|描述|
|---|---|
|bin|命令行脚本|
|cache|缓存访问层（redis)|
|config|项目配置文件|
|controllers|控制器|
|cron|定时任务|
|doc|项目文档（安装部署文档、接口文档、注释文档）|
|example|项目示例|
|lib|公共函数库|
|middlewares|中间件|
|models|models|
|node_modules|node_modules|
|proxy|代理层|
|public|静态文件|
|routes|路由|
|servers|服务启动入口|
|services|服务层|
|test|测试|
|tools|工具-与业务有关|
|utils|工具-与业务无关|
|.editorconfig|通用编辑器配置文件|
|.jscsrc|jscs配置文件|
|.jshintignore|js忽略文件列表配置文件|
|.jshintrc|jshint配置文件|
|gulpfile.js|gulp task|
|Makefile|Makefile|
|pm2-config.json|pm2启动配置文件|
|README.md|自描述文件|
|restart.sh|简单的服务重启脚本|
|start.cmd|windows开发启动脚本|
